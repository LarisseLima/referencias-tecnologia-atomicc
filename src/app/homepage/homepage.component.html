Sistema que reúne e consulta links de referências sobre os mais diversos assuntos de tecnologia. Mantido e atualizado de forma colaborativa.

Construído com o framework Angular na versão 9, com código aberto e disponível para melhoria contínua pela comunidade. Existem três formas de contribuir, veja abaixo.

Como contribuir

  • Clone o repositório
  • Crie sua branch.
  • Execute o comando 'npm install' para instalar as dependências.
  • Execute o comando 'npm run start' para rodar aplicação.
  • Adicione links úteis sobre qualquer assunto de tecnologia no arquivo 'referencias.json' adicionando palavras-chave para cada link.
  • Se quiser, você pode contribuir desenvolvendo melhorias ou novas funcionalidades.
  • Abra um pull request para a branch develop com suas modificações.
